Hexo方式

一、安装 Node.js

1. 下载安装

访问 Node.js 官网
下载 LTS 版本（长期支持版）
双击安装，一路点 Next，不需要改任何设置

2. 验证安装

打开命令提示符（CMD），输入：
node -v
npm -v

二、安装 Hexo

npm install -g hexo-cli
安装完成后验证：
hexo -v

三、创建博客项目

1. 初始化项目

cd C:\Users\25764
hexo init sxq-blog
cd sxq-blog
npm install

2. 目录结构说明

sxq-blog/
├── source/ # 存放文章、图片等
│ └── _posts/ # 博客文章放这里（.md 文件）
├── themes/ # 主题文件夹
├── _config.yml # 博客配置文件
├── package.json # 依赖管理
└── scaffolds/ # 文章模板

3. 本地预览

hexo server
简化命令
hexo clean && hexo g && hexo s
访问 http://localhost:4000

三、确认 deploy 配置

npm install hexo-deployer-git --save
打开 C:\Users\25764\sxq-blog\_config.yml，检查 deploy 部分是否是这样的：
deploy:
type: git
repo: https://github.com/sxq-happy/sxq-happy.github.io.git
branch: main

四、部署命令

hexo clean   # 清理缓存
hexo generate   # 生成静态文件（可简写为 hexo g）
hexo deploy    # 部署到 GitHub（可简写为 hexo d）
简化命令
hexo clean && hexo g && hexo d

五、日常修改流程

初始化
git init  
开始修改前，先确认当前状态干净
git status
本地预览测试
hexo clean && hexo g && hexo s
确认没问题后，提交版本
git add .
git commit -m "描述清楚这次改了什么，比如：调整首页Banner图片透明度"
部署上线
hexo d
